## **Task 1**

To change the color of heading of the caption section with the given color codes.
For Heading: #DE4839

### **After Update**

![](./ass9.1-after.png)

### **Project Solution**

```
const element = document.querySelector(".caption h1");
element.style.color = "#DE4839";
```
